# users-crud-hex
Spring Framework with Maven and Hexagonal Architecture

# Technologies
Docker  
MongoDB  
Postman

# Microservice Specification

**request body**

- firstName;  
- lastName;  
- birthDate;  
- email.  

**response body**

- id;  
- email;  
- fullName;  
- age.  

**rules**

- email is mandatory and exclusive;
- users under the age of 30 cannot be excluded.
